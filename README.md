# Computational Genomics and AI Portfolio Repository

This repository documents hands-on work in computational genomics, bioinformatics, technical writing, and introductory data analysis workflows using programming and open-source research tools.

The work in this repository focuses on building practical skills in:

- Genomics
- Artificial Intelligence
- Data Analysis
- Scientific Communication
- Reproducible Research
- Programming for Biological Data

The projects combine scientific reading, technical writing, coding exercises, exploratory analysis, and collaborative problem solving using real biological datasets and computational methods.

---

# Repository Goals

The purpose of this repository is to:

- Develop foundational bioinformatics skills
- Practice computational thinking in biology
- Learn scientific programming workflows
- Explore genomics datasets
- Build familiarity with AI-assisted biological research
- Document technical growth over time
- Create reproducible and well-organized scientific work

This repository serves as both a learning archive and a growing computational biology portfolio.

# Programming and Data Analysis

This repository contains coding exercises and computational workflows used to strengthen programming fundamentals for scientific computing.

Topics include:

- Variables and data structures
- Loops and conditionals
- Functions
- File handling
- Data visualization
- Spreadsheet processing
- Exploratory biological data analysis

Programming languages and tools may include:

- Python
- R
- Jupyter Notebooks
- Pandas
- OpenPyXL
- Git and GitHub
